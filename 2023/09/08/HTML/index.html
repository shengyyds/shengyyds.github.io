一、HTML基本知识

网页就是 HTML文件，后缀名： html 或 .htm

HTML 指的是超文本标记语言，它是用来描述网页的一种语言。HTML不是一种编程语言，而是一种标记语言。

超文本含义

1.他可以加入图片、声音、动画、多媒体等内容（超越了文本限制）。

2.它还可以从一个文件跳转到另一个文件，于世界各地主机的文件连接。

Web 标准

web 标准是由 W3c （万维网联盟）组织和其他标注话组织制定的一系列标准的集合。

构成：结构(HTML)、表现(css)、行为(Javascript)。

7.表单标签

使用表单的目的是为了收集用户信息

表单的组成：表单域、表单控件（也称单元素）和提示信息3部分组成。

1.表单域（在 form 的下一级）

属性 属性值 作用
action url地址 用于指定接收并处理表单数据的服务器程序的url地址。
method get/post 用于设置表单数据的提交方式，其取值为get或post。post 不会在 url 地址栏中显示
name 名称 用于指定表单的名称，以区分同一个页面中的多个表单域。
entype application/x-www-form-urlencoded； multipart/form-data ；text/plain 规定在发送表单数据之前如何让对其进行编码(在发送前编码所有的字符，默认的 / 不对字符编码。使用包含文件上传空间的表单时，必须使用该值 / 空格转换为“+” ，但是不对特殊字符编码)

表单的同步提交

缺点：提交后，会直接跳转到提交页面，原有的数据就会被清空。

解决：表单采集数据，用 Ajax 负责将数据提交到服务器。

2.表单控件（表单元素）

属性值 描述（这些东西都要在 type 里面）
radio 定义单选按钮
checkbox 定义复选框（多选按钮）
placeholder 用于未输入时提示的文字，一旦有输入，立刻消失
password 定义密码字段。该字段中的字符被掩码
text 定义单行的输入字段，用户可在其中输入文本。默认宽度为20个字符
submit 定义提交按钮。会把表单数据发送到服务器(后面加 value 可以将 “提交” 字样修改)
reset 定义重置按钮。会清除表单中的所有数据
button 定义可点击按钮(如：获取验证码)，多数情况下，用于通过JavaScript启动脚本
file 定义输入字段和“浏览”按钮，可供文件上传
image 定义图像形式的提交按钮
hidden 定义隐藏的输入字段
date 可以输入/选择日期。可以用 min 更改最小值
number 输入数字。可以用min 和 max 更改范围
属性名称 属性值 定义(都要在 input 里面)
name 属性由用户自定义 定义input元素的名称。
value 属性由用户自定义 定义 input元素的值（表单中的值），用于提交给后台的数据
checked checked 规定此 input元素首次加载时应当被选中。(打开页面时自动勾选)
maxlength 正整数 规定输入字段中的字符的最大长度。

注意：

没写name 的时候，只写 radio 可以多选，当写了 name （相同的时候）的时候就只能单选了

value相当于数据绑定 你输入的数据更改的就是value里面的值 提交form就是把value提交出去.

name和value是每个表单元素都有的属性值，主要给后台人员使用.

name表单元素的名字,要求单选按钮和复选框要有相同的name值.

checked属性 详解
注意：当元素中有checked属性时，其值无论是什么，都是被选中状态，那怎么才能让其不被选中呢，就是用jquery或js代码实现
　　1、html中的checked属性。仔细研究下会发现一个很怪异的现象。
复选框里只要有checked属性，不管是否为其赋值，结果为空或true或false或任意值，均为选中状态。
　　2、利用javascript操作checked来控制复选框是否选中
　　要使其不选中，即设置checked属性值为false。

　链接：https://www.cnblogs.com/mark5/p/11023851.html

属性 值 说明
