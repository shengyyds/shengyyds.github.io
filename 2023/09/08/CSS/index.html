一、简单介绍

1.简介

CSS 指的是：Cascading Style Sheets

CSS 的主要使用场景就是美化网页，布局页面的。是 “ 层叠样式表 ” 的简称，也可以称之为 CSS 样式表或级联样式表。

CSS 也是一种标记语言，主要用于设置 HTML 的文本内容、图片的外形以及版面的布局和外观显示样式。

CSS 最大价值:由 HTML 专注去做结构呈现，样式交给 CSS，即结构( HTML )与样式( CSS ）相分离。

2.语法规范

CSS 规则由两个主要的部分组成：选择器以及一条或多条声明。

选择器是用于指定 CSS 样式的 HTML 标签，花括号内是对该对象设置的具体样式

属性和属性值以 “键值对(英文冒号 : )” 的形式出现。多个“键值对”之间用央又“;”进行区分。

CSS 主要写在 内部的 里面

3.代码规范

快捷键：shift+alt+F 格式化代码

样式选择器，属性名，属性值关键字全部使用小写字母，特殊情况除外。

属性值前面，冒号后面，保留一个空格。选择器（标签）和大括号中间保留空格。

三、选择器

1.选择器的作用

选择器（选择符）就是根据不同需求把不同的标签选出来这就是选择器的作用。简单来说，就是选择标签用的。

2.选择器分类

选择器分为基础选择器和复合选择器。

3.基础选择器

基础选择器是由单个选择器组成的，基础选择器又包括：标签选择器、类选择器、id 选择器和通配符选择器。

1.标签选择器

作用：标签选择器可以把某一类标签全部筛选出来。这样便于快速为页面中同类型的标签统一设置样式。

2.类选择器

作用：可以差异化选择不同的标签，单独选择一个或者几个标签。

语法：要使用 class 属性

记忆口诀：样式点定义，结构类(class)调用，一个或多个，开发最常用。

类选择器使用 “.”( 英文点号 ) 进行标识，后面紧跟类名（类名中间可以使用 “- “），类名不要用 div、span， 这种标识符。

类名尽量使用英文字母来表示，不要使用纯数字、中文等命名。

1.多类名

多类名，顾名思义，就是使用多个类名，但是在多个类名中间要用空格隔开

3.id 选择器

HTML 元素以 id 属性来设置 id 选择器，CSS 中 id 选择器以 “#” 来定义。

样式 # 定义，结构 id 调用，而且只能调用一次

4.通配符选择器

CSS 中，通配符选择器使用 ” ***** “ 定义，他表示选取页面中所有标签。

5.选择器优先级

id 选择器 > 类选择器 > 标签选择器 > 通配符选择器

6.属性选择器

可以根据某个属性的属性值来选择标签

六、CSS 的三大特性

层叠性、继承性、优先级

1.层叠性

若相同选择器给设置相同的样式，此时靠近结构的样式会覆盖其他样式，并不是全部层叠掉，而是将上面与下面相同样式的覆盖掉，不相同的不会层叠掉。

2.继承性

子标签会继承父标签中的样式

只会继承 text- ,font- ,line-（跟文字有关的）和 color 这些属性。

3. 初始值

可以使用 intial 关键字显示重置为初始值

3.优先级

当同一个元素有多个选择器指定，会有优先级的产生。若选择器相同，则执行层叠性，否则，按权重划分：

继承 = 通配符 < 元素选择器 < 类选择器 = 伪类选择器 < ID选择器 < 行内样式 style=“ ” < !important。

继承的权重为 0；

复合选择器会有权重叠加问题，但是不会进位，权重低的一直低。

八、网页布局

网页布局的本质：用 CSS 来摆放盒子

传统网页布局的三种方式：标准流、浮动、定位。

1.标准流

例如：块级元素、行内元素。
就是按照规定好的默认方式进行排列，是最基本的布局方式。一个完整的网页是用三种方式同时构成的

2.浮动

网页布局第一准则∶多个块级元素纵向排列找标准流，多个块级元素横向排列找浮动。

应用：
实现叠加效果：一个加浮动，一个不加，不加的会跑到浮动盒子下面。

为了约束浮动元素位置,我们网页布局一般采取的策略是:
先用标准流的父元素排列上下位置,之后内部子元素采取浮动排列左右位置.符合网页布局第一准侧.

特性：
1.脱离标准普通流的控制(浮）移动到指定位置(动)，(俗称脱标)，不再保留原先的位置

2.如果多个盒子都设置了浮动，则它们会按照属性值一行内显示并且顶端对齐排列。

3.浮动元素会具有行内块元素特性。
任何元素都可以浮动。不管原先是什么模式的元素，添加浮动之后具有行内块元素相似的特性，可以直接给宽度和高度。

4.如果块级盒子没有设置宽度，默认宽度和父级一样宽，但是添加浮动后，它的大小根据内容来决定

注意：

1.浮动的盒子只会影响浮动盒子后面的标准流不会影响前面的标准流.

2.注意∶浮动的元素是互相贴靠在一起的(不会有缝隙），如果父级宽度装不下这些浮动的盒子，多出的盒子会另起一行对齐。

3.一个元素浮动了，理论上其余的兄弟元素也要浮动。

清除浮动

因为当一个页面可能处于不断地增加或者减少的状态，或者处于灵活变换的状态，内容可多可少的，不方便给高度时。

清除浮动的原因：由于父级盒子很多情况下，不方便给高度，但是子盒子浮动又不占有位置，最后父级盒子高度为 0 时，就会影响下面的标准流盒子。

清除浮动的本质：清除浮动元素造成的影响。

清除浮动的策略：闭合浮动。

清除浮动之后，父级就会根据浮动的子盒子自动检测高度。父级有了高度，就不会影响前面的标准流了

3. 定位

使用场景：
浮动可以让多个块级盒子一行没有缝隙排列显示，经常用于横向排列盒子。
定位则是可以让盒子自由的在某个盒子内移动位置或者固定屏幕中某个位置，并且可以压住其他盒子。

1. 定位组成

定位：将盒子定在某个固定的位置。
定位 = 定位模式 + 边偏移。
定位模式用于指定一个元素在文档中的定位方式，边偏移则决定了该元素的最终位置。

1. 定位模式

定位模式决定元素的定位方式，它通过 CSS 的 position 属性来设置，其值分为：static（静态定位）、relative（相对定位）、absolute（绝对定位）、fixed（固定定位）、sticky（粘性定位）。

静态定位

特性：按照标准流方式摆放盒子

相对定位

特性：
相对定位是元素在移动位置的时候，是相对于它原来的位置来说的（top: 100px 真实的情况是盒子向下移动 100px。）
原来在标准流的位置继续占有，后面的盒子仍然以标准流的方式对待它，后面的元素不脱标，继续保留原来位置。

不切当使用会导致：某一个盒子在增加相应移动后，距离原本应该到达的位置有一部分误差，这部分原因是因为，相对移动后的元素还占用原有位置。

解决方法：尽量使用 “子绝父相” 来修饰。

绝对定位

绝对定位是元素在移动位置的时候，是相对于它祖先元素来说的（拼爹型)。
特性：
如果没有祖先元素或者祖先元素没有定位，则以浏览器为准定位（Document文档)；
若父级元素没定位，就向上寻找有定位的上级，若都没有才吧浏览器当作相对对象。
不占有位置。

子绝父相

定义：子元素使用绝对定位，父元素必须使用相对定位

固定定位

是元素固定在浏览器可视区的位置，跟父元素没有关系，不随网页滚动而滚动。

特殊应用：
固定在版心右边。
方法：让固定定位的盒子先走到版心的位置，然后再走版心盒子宽度的一半。

粘性定位

固定定位和相对定位的结合体。
特性：
在页面的一部分区域内固定，在另一区域内不固定；
占有原来的位置，不脱标；
必须添加边偏移。

绝对定位居中

加了绝对定位的盒子不能使用 margin: auto; 来实现水平居中
方法：

定位的特殊属性

行内元素添加绝对定位或者固定定位，可以直接给宽度和高度；
块级元素如果给绝对定位或者固定定位，而不给宽度和高度，默认大小是内容的大小；

脱标的盒子不会出发外边距塌陷

浮动、绝对定位元素都不会出发外边距合并问题。

绝对定位会压住盒子

浮动元素之会压住盒子，不会压住里面的图片和文字，因为浮动原本的目的是用来实现文字环绕效果，但绝对定位会压住文字；

1.新增选择器

1. 属性选择器

定义：可以根据特定属性来选择选择

注意：属性选择器也应该注意权重的问题
类选择器、属性选择器、伪类选择器的权重为 0。

2. 结构伪类选择器

定义：主要根据文档结构来选择器元素，常用于根据选择器里面的子元素

选择符 简介
E:first-child 匹配父元素中的第一个子元素E
E:last-child 匹配父元素中最后一个E元素
E:nth-child(n) 匹配父元素中的第n个子元素E
E:first-of-type 指定类型E的第一个
E:last-of-type 指定类型E的最后一个
E:nth-of-type(n) 指定类型E的第n个

注意：

  1. 冒号前面一定不要有空格，否则会不渲染。
  2. 渲染的时候，先寻找 nth-child(n) 对应的第几个元素，在把找到的和 nth-child(n) b前面想找到的进行匹配，若匹配，就进行渲染，否则什么也不发生。

拓展

:nth-child(n) 的括号里面可以写：数字、公式（只能含有 n 的公式）和关键字（ even 偶数、odd 奇数）。

写公式的时候，公式从 n=0 开始算起，如果得到的数不符合就继续往下运行，直到没有对应的元素可以选择。

注意： 他们能创建一个元素，该元素属于行内元素，但是这个元素在文档树中是找不到的，所以称之为伪元素。
