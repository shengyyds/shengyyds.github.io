一、编程语言

编程语言和标记语言

编程语言分为：机器语言、高级语言和汇编语言；

汇编语言和机器语言的本质是相同的，都是直接对硬件操作，只不过使用的是缩写的英文。

高级语言是适合人类学习的语言。

区别： 编程语言具有很强的逻辑和行为能力的指令，这是主动的；标记语言不用想计算机发出指令，常用于格式化和链接，标记语言的存在是用来被读取的，是被动的。

七、预解析和代码执行

引擎运行分为两步：预解析和代码执行

  1. 引擎会把 JavaScript 里面所有的 var 和 function 提升到当前作用域的最前面。
  2. 代码执行是从上往下执行的。

变量提升：是指把所有的变量声明提升到当前的作用域最前面，不提升赋值操作。

函数提升：是指把所有的函数声明提升到当前的作用域最前面，不调用函数。

  3. addEventListener(type, listener,useCapture 第三个参数如果是true，表示在事件捕获阶段调用事件处理程序;如果是false (不写默认就是false )，表示在事件冒泡阶段调用事件处理程序。
  4. 捕获阶段会从大到小，从外由内，当点击子元素时，父元素事件也会被调用，先执行父元素函数
  5. 有些事件是没有冒泡的，比如onblur、onfocus、onmouseenter、onmouseleave

事件对象只有有了事件才会存在，他是系统给我们自动创建的不需要我们传递参数

事件对象是我们事件的一系列相关数据的集合跟事件相关的

十三、BOM

1. 定义

BOM (Browser Object Model) 即浏览器对象模型，它提供了独立于内容而与浏览器窗口进行交互的对象，其核心对象是 window

BOM 缺乏标准，兼容较差

window 对象是浏览器的顶级对象，具有双重角色

  1. 他是 JS 访问浏览器窗口的一个接口
  2. 他是一个全局变量，定义在全局作用域中的变量、函数 都会变成 window 对象的属性和方法

window.name 是一个特殊属性，使用时尽量不要使用他

4. 执行机制

JavaScript 的一特点就是单线程，同一个时间就只能干一件事，因为这是这门语言的使命所致：处理页面中用户的交互，以及操作DOM 而诞生的。

单线程就意味着，所有任务需要排队，前一个任务结束，才会执行后一个任务。这样所导致的问题是∶如果JS执行的时间过长，这样就会造成页面的渲染不连贯，导致页面渲染加载阻塞的感觉。

为了解决这个问题，利用多核CPU的计算能力，HTMLS提出 Web Worker标准，允许JavaScript脚本创建多个线程。于是，JS中出现了同步和异步。

同步是按照程序的顺序执行程序，异步却可以跳过时间较长的执行时间短的。

问题：

JS 的异步是通过回调函数实现的

分为：普通事件、资源加载、定时器

执行步骤：

  1. 先执行执行栈中的同步任务
  2. 碰到回调函数，就先放到任务队列中，不执行
  3. 一旦执行栈中的同步任务执行完毕，系统就会按任务队列读取异步任务

事件循环：主线程不断重复获得任务、执行任务、再获取任务、在执行的机制

5. location 对象

window 对象给我们提供了一个location属性用于获取或设置窗体的 URL，并且可以用于解析 URL。因为这个属性返回的是一个对象，所以我们将这个属性也称为location对象。

URL

又名：统一资源定位符，他是互联网上标准资源的地址。互联网上的每个文件都有一个唯一的 URL ，它包含的信息指出文件的位置以及浏览器怎么处理他

URL的一般语法格式为︰
protocol: / / host [ : port]/path/ [ ?query]#fragment

例如：http: / / www.itcast.cn/index.html ?name=andy&age=18#link

组成 说明
protocol 通信协议；常用的 http,ftp,maito等
host 主机 ( 域名) ；www.itheima.com
port 端口号可选，省略时使用方案的默认端口如http的默认端口为 80
path 路径由零或多个’/符号隔开的字符串，一般用来表示主机上的一个目录或文件地址
query 参数以键值对的形式,通过&符号分隔开来
fragment 片段 # 后面内容常见于链接锚点

1. 属性

location对象属性 返回值
location.href 获取或者设置整个URL
location.host 返回主机(域名)www.itheima.com
location.port 返回端口号如果未写返回空字符串
location.pathname 返回路径
location. search 返回参数
location.hash 返回片段#后面内容常见于链接锚点
location.replace 修改当前 url，不记录浏览历史
location.assign 修改当前 url，记录浏览历史，可后退
location.reload 重新加载页面，等于 F5

1. 元素偏移量 offset 系列

offset 翻译过来就是偏移量，我们使用此系列相关属性可以动态的得到该元素的位置（偏移）、大小等。

  1. 获得元素距离带有定位父元素的位置
  2. 获得元素自身的大小
  3. 返回的值都不带有单位
offset系列属性 作用
element.offsetParent 返回作为该元素带有定位的父级元素如果父级都没有定位则返回 body
element.offsetTop 返回元素相对带有定位父元素上方的偏移
element.offsetLeft 返回元素相对带有定位父元素左边框的偏移
element.offsetWidth 返回自身包括padding+边框+内容区的宽度，返回数值不带单位
element.offsetHeight 返回自身包括padding、边框、内容区的高度，返回数值不带单位

offset 和style 的区别

offset style
可以得到任意样式表中的值 只能得到行内样式表中的样式值
得到的数值没单位 带单位
盒子 width = 内容 + 边框 + padding 实际大小
只读属性，不能修改 可以修改
适合获取数值 适合修改数值使用

本质区别： offset 不能改变元素的属性值，style 可以

element.clientHeight 返回自身包括padding 、内容区的高度，不含边框，返回数值不带单位
